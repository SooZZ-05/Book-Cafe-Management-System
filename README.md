# Book Cafe Management System

A Java-based console application developed for the **BACS2023 Object-Oriented Programming** assignment. This system was designed for **NOVA Book Café** to simplify café operations and provide a smooth customer experience through book browsing, food ordering, cart management, checkout, and order tracking.

## Features

- User registration and login
- Forgot password functionality
- Book searching by title
- Book sorting by:
  - Title
  - ISBN
  - Author
  - Rating
  - Publication year
- View food items
- Add and remove items from cart
- Checkout process
- View order history
- Membership registration with discount benefits
- Ratings and comments for purchased books

## My Contribution

My main contributions to this project were:

- **Checkout process**
- **Viewing order history**

These modules included payment amount calculation, payment validation, membership fee handling, discount application, and displaying previous orders with purchased items, date/time, and total amount paid.

## Payment Validation
Validation includes:
- Visa card format checking
- **Luhn algorithm** for credit card validation
- Expiry date validation
- CVV validation
- Malaysian phone number and 6-digit password validation
